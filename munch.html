Munch!

Eat the fruits and not anything else ~!

A web-app game! I wanted to recreate a game that I played as a child. So, I decided to program the classic game where you have to collect items falling and earn points from it.


I started planning using the Agile Game Development, where I had to map the game stories. The game development started with just collecting the items and earning points. To make the game more fun and interesting, I decided to add other random tools that you have to avoid. My classmates then gave me the suggestion to implement lives and score points to make the game more interactive :D

Technical Skills Used:

  • CSS
  • Javascript
  • DOM Manipulation
